# Installing Ads Magic on WooCommerce

1. Log in to your **WordPress Admin**.
2. Go to **Plugins > Add New**.
3. Search for **"Ads Magic"**.
4. Click **Install Now**, then **Activate**.
5. Once active, you will see a new **"Ads Magic"** menu item in your sidebar.
6. Click it to run your first **Health Check**.
