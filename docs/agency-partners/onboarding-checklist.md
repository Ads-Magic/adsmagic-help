# Client Onboarding Checklist

Use this checklist to ensure every new client setup is perfect.

### Phase 1: The "Spark" (Installation)
- [ ] Install **Ads Magic Plugin** on client's WordPress site.
- [ ] Run **Health Check**. Resolve any Red errors (GTINs, Images).
- [ ] Connect **Google Merchant Center** account.

### Phase 2: The "Alchemist" (Tracking)
- [ ] Connect **Ads Magic Cloud** (using your Agency Master Key or Client's Login).
- [ ] Enable **Server-Side Tracking (CAPI)**.
- [ ] Input **Meta Pixel ID** and **Access Token**.
- [ ] Verify events are firing in Facebook Events Manager.

### Phase 3: The "Archmage" (Profit)
- [ ] Input **COGS** for all products (Bulk Edit or CSV).
- [ ] Configure **Custom Labels** (High/Low Margin).
- [ ] Update **Google Ads** to import the "POAS" custom metric.

### Phase 4: Handoff
- [ ] Enable **White Label** mode (if applicable).
- [ ] Send welcome email to client with their dashboard login.
